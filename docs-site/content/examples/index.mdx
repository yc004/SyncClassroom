# 示例总览

本页收集当前最常用的课件交互模式。示例默认写在 Zip `.lume` 的 `slides/*.tsx` 页面模块中，并通过 `@lumesync/course-sdk` 使用课堂上下文。

## 可用示例

| 示例 | 解决的问题 |
| --- | --- |
| [同步变量](/examples/sync-var) | 教师端控制状态，学生端跟随变化 |
| [学生提交](/examples/submission) | 学生把答案或文件内容提交给教师端 |
| [投票组件](/examples/vote) | 教师端发起课堂即时投票并统计 |

## 推荐 .lume 结构

```text
course.lume
├─ manifest.json
├─ slides/
│  ├─ SyncVarDemo.tsx
│  ├─ SubmissionDemo.tsx
│  └─ VoteDemo.tsx
└─ assets/
```

## 注意事项

- 示例中的页面组件需要在 `manifest.json` 的 `pages` 数组中登记。
- `useSyncVar` 依赖教师端开启 `syncInteraction`。
- `submitContent` 只能在学生端使用。

# 机房视图使用指南

## 概述

机房视图（ClassroomView）是教师端用于管理学生座位的工具，支持：
- 查看学生在线状态
- 配置学生姓名和学号
- 拖拽调整座位布局
- 导入/导出座位列表
- 多班级管理

## 核心功能

### 1. 座位配置

每个座位包含以下信息：
- **IP 地址**：学生终端的 IP 地址（必填）
- **学生姓名**：学生姓名（可选，用于问卷等场景）
- **学号**：学生学号（可选，用于问卷等场景）
- **行号**：座位所在行（1 起始）
- **列号**：座位所在列（1 起始）

### 2. 学生信息的重要性

**为什么需要配置学生姓名和学号？**

当学生提交问卷、作业等内容时，服务器会根据学生 IP 从座位表中查询学生信息，并自动填充到提交数据中。这样可以：
- ✅ 自动关联学生身份，无需学生手动填写
- ✅ 防止学生伪造身份
- ✅ 统一管理学生信息

**示例：问卷提交**

学生提交问卷后，CSV 文件会自动包含：
```
提交时间,学生IP,学生姓名,学号,题目1,题目2,状态
2024-03-25T10:30:00.000Z,192.168.1.101,张三,20230001,A,B,submitted
```

## 操作指南

### 1. 手动添加座位

1. 点击右上角"手动添加"按钮
2. 填写以下信息：
   - **IP 地址**：学生终端的 IP（如 `192.168.1.101`）
   - **学生姓名**：学生姓名（如 `张三`）
   - **学号**：学生学号（如 `20230001`）
   - **行**：座位行号
   - **列**：座位列号
3. 点击"添加"

### 2. 自动导入在线学生

1. 确保学生端已连接到课堂
2. 点击"自动导入"按钮
3. 系统会自动将当前在线学生添加为座位
4. 然后手动编辑每个座位，填写姓名和学号

### 3. 编辑座位信息

**网格视图：**
- 双击座位的名称区域
- 在弹出的编辑窗口中修改姓名和学号
- 点击"保存"完成

**列表视图：**
- 点击学生姓名
- 在弹出的编辑窗口中修改姓名和学号
- 点击"保存"完成

### 4. 拖拽调整座位位置

- 在网格视图中，按住座位拖拽到目标位置
- 可以与另一个座位交换位置
- 拖拽到空位置则直接移动

### 5. 删除座位

- 将鼠标悬停在座位上
- 点击右上角的删除按钮（×）

### 6. 导入座位列表

支持 CSV 和 JSON 格式导入。

**CSV 格式：**

创建一个 CSV 文件，格式如下：
```csv
ip,名称,学号,行,列
192.168.1.101,张三,20230001,1,1
192.168.1.102,李四,20230002,1,2
192.168.1.103,王五,20230003,1,3
```

导入步骤：
1. 点击"更多" → "导入列表"
2. 选择 CSV 或 JSON 文件
3. 确认导入

**JSON 格式：**

```json
{
  "version": 2,
  "classroomId": "classroom-123",
  "classroomName": "高一1班",
  "podiumAtTop": true,
  "seats": [
    {
      "ip": "192.168.1.101",
      "name": "张三",
      "studentId": "20230001",
      "row": 1,
      "col": 1
    },
    {
      "ip": "192.168.1.102",
      "name": "李四",
      "studentId": "20230002",
      "row": 1,
      "col": 2
    }
  ]
}
```

### 7. 导出座位列表

**导出为 CSV：**
1. 点击"更多" → "导出列表 (CSV)"
2. 保存文件，格式：`classroom-seats-YYYYMMDD-HHMM.csv`

**导出为 JSON：**
1. 点击"更多" → "导出当前班级 (JSON)"
2. 保存文件，格式：`classroom-班级名-YYYYMMDD-HHMM.json`

### 8. 下载模板

如需快速开始，可下载 CSV 模板：
1. 点击"更多" → "下载 CSV 模板"
2. 按模板格式填写后导入

### 9. 多班级管理

**切换班级：**
1. 点击左上角班级名称
2. 在下拉菜单中选择其他班级

**创建新班级：**
1. 点击班级下拉菜单
2. 点击"新建"
3. 输入班级名称（如"高一1班"）
4. 点击"创建"

**删除班级：**
1. 点击班级下拉菜单
2. 鼠标悬停要删除的班级
3. 点击右侧的删除按钮（垃圾桶图标）
4. 确认删除（注意：删除后无法恢复）

**重命名班级：**
- 双击班级名称即可重命名

## 视图模式

### 网格视图

- 直观的座位布局显示
- 支持拖拽调整位置
- 显示在线/离线状态
- 显示最近告警信息（退出全屏、切换页面等）

### 列表视图

- 表格形式列出所有座位
- 便于查看完整信息
- 支持点击编辑
- 显示序号、状态、IP、姓名、学号、行列、告警

## 状态说明

**在线状态：**
- 🟢 绿色圆点：学生在线
- ⚫ 灰色圆点：学生离线

**告警信息：**
- 🟠 压缩图标：学生退出了全屏模式
- 🔒 眼睛斜杠图标：学生切换了页面（不在当前课件）
- 🟢 用户加号：学生上线
- ⚪ 用户减号：学生离线

## 讲台位置

- 默认讲台在上方
- 点击"讲台在上/下"按钮可切换讲台位置
- 便于根据教室实际布局调整

## 数据同步

座位表数据会自动同步到服务器（`public/data/classroom-layout-v1.json`），用于：
- 学生提交问卷时自动填充学生信息
- 其他需要学生身份的场景

## 最佳实践

1. **课前准备**：提前导入座位列表，配置好姓名和学号
2. **定期备份**：导出 JSON 文件备份座位配置
3. **多班级管理**：不同班级使用独立的座位配置
4. **IP 映射**：确保 IP 地址与学生终端一一对应
5. **及时更新**：学生转班或换座位时及时更新

## 常见问题

**Q: 学生提交的问卷中姓名和学号为空？**

A: 检查以下几点：
1. 是否在机房视图中正确配置了该 IP 对应的姓名和学号
2. IP 地址是否正确（注意 IPv4 和 IPv6 的区别）
3. 学生提交的 IP 是否与座位表中的 IP 完全一致

**Q: 如何批量更新学生信息？**

A: 使用导入/导出功能：
1. 导出当前座位列表为 CSV
2. 在 Excel 或文本编辑器中批量修改
3. 重新导入（覆盖模式）

**Q: 座位信息会自动同步到学生端吗？**

A: 不会。座位信息仅用于服务器端查询和填充学生信息，学生端不会看到座位表内容。

**Q: 如何处理 IP 变化的情况？**

A: 如果学生终端 IP 发生变化，需要：
1. 删除旧 IP 的座位
2. 添加新 IP 的座位
3. 重新配置姓名和学号

## 查看学生提交文件

在教师端主界面，点击"查看学生文件"按钮可以浏览和下载学生提交的所有文件。

### 功能说明

- **按课程分类**：左侧显示所有课程，点击切换查看对应提交文件
- **文件预览**：支持在线预览 JSON、CSV、TXT 等文件格式
- **批量下载**：勾选多个文件后可打包为 ZIP 下载
- **文件信息**：显示班级、日期、大小等详细信息

### 使用方法

**查看文件**
1. 点击"查看学生文件"按钮
2. 在左侧选择课程
3. 点击文件可预览内容
4. 鼠标悬停可下载单个文件

**批量下载**
1. 勾选需要下载的文件
2. 点击"批量下载"按钮
3. 系统自动打包并下载 ZIP 文件

**支持的预览格式**

| 格式 | 预览方式 |
|-----|---------|
| JSON | 智能识别问卷、留言等格式 |
| CSV | 表格形式展示 |
| TXT | 纯文本显示 |
| 其他 | 仅支持下载 |

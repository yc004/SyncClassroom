# 03 机房视图与座位表

## 1. 机房视图的用途

机房视图用于把“在线学生”、“座位位置”和“学生身份信息”统一到同一张视图中。

它主要解决三个问题：

- 知道谁在线、谁离线
- 把学生 IP 和学生姓名、学号建立映射
- 让提交数据能自动带出学生身份

## 2. 机房视图能做什么

- 手动添加座位
- 导入在线学生
- 编辑姓名与学号
- 拖拽调整位置
- 导入和导出座位表
- 管理多个班级

## 3. 为什么它很重要

学生提交、问卷和课堂活动常常只带 IP。教师端通过座位表可以把这些数据补全为可读的学生身份信息。

## 4. 使用建议

- 课前先维护好座位表
- 用统一的 IP 与学生身份映射
- 班级切换时检查当前班级是否正确
- 定期导出座位表作为备份

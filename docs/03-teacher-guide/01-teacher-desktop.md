# 01 教师端使用说明

## 1. 教师端负责什么

教师端是整套系统的控制中心。它负责：

- 启动课堂服务
- 管理课件和目录
- 选择课程并开始课堂
- 控制翻页、同步、全屏和互动
- 查看学生在线状态与提交结果

## 2. 上课前准备

在正式上课前，建议确认以下事项：

- 教师机和学生机在同一局域网
- 教师端可以正常启动
- 学生端知道教师机 IP 地址
- 课程已经导入并可在教师端看到

## 3. 一次完整上课流程

1. 启动教师端
2. 进入课程选择界面
3. 选择或导入课件
4. 确认学生端已连接
5. 开始授课并控制翻页
6. 按需开启投票、互动或标注
7. 课后查看学生提交并下载结果

## 4. 常用功能

- 课程选择
- 翻页同步
- 教师设置同步
- 标注绘制与清除
- 学生在线人数查看
- 学生提交浏览和批量下载

## 5. 常见问题

### 学生端连不上

先检查：

- 教师端是否已启动
- 教师机 IP 是否正确
- 是否处于同一局域网
- 端口 `3000` 是否被防火墙拦截

### 学生端不同步

先检查：

- 当前课程是否已选中
- 是否开启了跟随翻页
- 学生端是否已连到正确教师地址

# 术语表

## 根仓

指当前 `SyncClassroom` 仓库。它不是业务主仓，而是系统总览仓。

## 子仓库

指 `repos/core`、`repos/teacher`、`repos/student`、`repos/editor-plugin` 四个真实业务仓库。

## 教师端

教师桌面应用及其本地服务。它同时承担课堂控制入口和内容真源职责。

## 学生端

学生桌面应用。负责连接教师端服务、接收同步状态、渲染课件和提交互动结果。

## 核心运行时

指连接控制、课堂状态同步、Socket 事件总线和渲染运行时这一层能力。它不负责业务文件管理。

## 数据面

指课程、文件、布局、提交、导入导出等业务数据接口。当前由教师端负责。

## 控制面

指选课、翻页、互动、投票、标注、在线状态等课堂控制和同步事件。

## 课件真源

指课件文件、课程目录、布局和提交文件的唯一管理者。当前为教师端。

## 渲染引擎

指用于渲染课件页面和组件的运行时能力。

## `.lume`

项目内使用的课件文件或课件工程格式。编辑器插件主要围绕这一格式提供本地预览能力。

## 子模块

Git `submodule`。根仓通过它引用四个真实子仓库，而不是复制一份代码到本地。

## 子模块指针

根仓记录的某个子仓库提交引用。子仓有新提交后，根仓只有更新指针并提交，才会反映到根仓历史中。

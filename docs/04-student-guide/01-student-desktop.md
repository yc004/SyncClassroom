# 01 学生端使用说明

## 1. 学生端负责什么

学生端的职责很明确：

- 连接教师端
- 接收同步课件
- 跟随翻页
- 参与投票和互动
- 提交课堂结果

## 2. 学生端需要知道什么

学生端通常只需要正确配置：

- 教师机 IP
- 端口，默认 `3000`

## 3. 启动后的典型状态

学生端常见界面有三种：

- 连接失败页
- 等待教师选课页
- 正式课堂展示页

## 4. 常见使用流程

1. 启动学生端
2. 配置教师机地址
3. 等待教师开始课程
4. 进入课堂并自动跟随同步
5. 按要求参与互动或提交

## 5. 连接失败时先查什么

- 教师端是否已启动
- 教师机 IP 是否正确
- 学生端与教师机是否在同一局域网
- 教师机端口是否被防火墙拦截

## 6. 学生端和教师端的关系

学生端不是内容真源。它不管理课程目录，也不保存业务文件。它只是课堂内容的消费端和互动参与端。

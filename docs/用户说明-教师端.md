# 萤火课堂（LumeSync）教师端使用说明（零基础）

这份说明写给第一次使用萤火课堂的老师/助教：不需要懂编程，照着做即可开始上课。

你将学会三件事：

- 让学生端“连上来”（最关键）
- 选课件并开始授课（翻页、同步、强制全屏）
- 课上标注（画笔/清空）与课件导入导出

---

## 1. 教师端是做什么的？

教师端是“主控台”，用于：

- 启动课堂（在本机开启课堂服务）
- 选择课件并开始授课
- 翻页、同步学生端画面
- 课堂设置（强制全屏、同步跟随、缩放、管理员密码等）
- 查看在线学生与学生操作日志
- 课上标注（绘制/清空）

---

## 2. 5 分钟快速开始（推荐先看这一节）

按下面清单做完，基本就能开课：

1. 打开教师端软件（保持运行）
2. 在教师端看到“课程选择界面”
3. 记下教师机 IP（例如 `192.168.1.100`）
4. 让学生端把“教师机 IP”改成你的 IP（同一个局域网）
5. 在教师端选择课程，进入课堂

![图 1：教师端“课程选择”界面（示例）](images/user-guide/teacher/01-course-selector.png)

## 3. 上课前准备（非常重要）

### 3.1 确认网络（学生连不上 90% 都是这里）

要让学生连接成功，必须满足：

- 教师机和学生机在**同一个局域网**（同一个路由器/Wi-Fi）
- 教师机上课时保持开机，且教师端软件保持运行

![图 2：同一局域网示意图（教师机与学生机在同一 Wi‑Fi/网段）](images/user-guide/teacher/02-lan.png)

### 3.2 确认端口/防火墙

萤火课堂默认使用端口 `3000`（一般不用改）。如果学生机连接不上：

- 先确认教师端已打开并能看到课堂界面
- 再检查 Windows 防火墙/安全软件是否拦截了端口 `3000`

![图 3：Windows 防火墙允许应用/端口（示例）](images/user-guide/teacher/03-firewall.png)

### 3.3 找到教师机 IP（一定要会）

学生端需要填的是“教师机 IP”，不是 `localhost`。

方法 A（推荐）：命令行查看

1. 按 `Win + R`
2. 输入 `cmd` 回车
3. 输入 `ipconfig` 回车
4. 找到“IPv4 地址”，例如 `192.168.1.100`

![图 4：用 ipconfig 查看 IPv4 地址（示例）](images/user-guide/teacher/04-ipconfig.png)

方法 B：系统设置查看

1. 打开“设置” → “网络和 Internet”
2. 找到当前连接的 Wi‑Fi/以太网
3. 查看“IPv4 地址”

![图 5：Windows 设置里查看 IPv4 地址（示例）](images/user-guide/teacher/05-ip-settings.png)

## 4. 安装与启动

### 4.1 安装

双击 `LumeSync-Teacher-Setup-*.exe` 安装包，按向导完成安装。

![图 6：教师端安装向导（示例）](images/user-guide/teacher/06-install.png)

### 4.2 启动

打开“萤火课堂 教师端”。启动后软件会在本机开启课堂服务，并进入教师端主界面。

提示：上课期间教师端不要关闭。教师端一关，学生端就会掉线并进入重连。

---

## 5. 开始上课（最常用）

### 5.1 选择课程

在“课程选择”界面，点击一个课程卡片即可进入课堂。

如果你刚拷贝了新课件但列表里没看到：

- 点击“刷新课程列表”

![图 7：点击课程卡片进入课堂（示例）](images/user-guide/teacher/07-click-course.png)

### 5.2 学生端如何进入课堂？

学生端需要知道教师机的局域网 IP（例如 `192.168.1.100`）。一般学校机房会提前统一设置；如果临时更换教室或网段，学生端需要修改为你当前的教师机 IP。

教师端界面会提示：

- 教师端：`http://localhost:3000`
- 学生端：`http://<LAN-IP>:3000`

这里的 `<LAN-IP>` 就是要告诉学生/机房管理员设置的地址。

![图 8：教师端显示“学生端访问地址”的位置（示例）](images/user-guide/teacher/08-student-url.png)

## 6. 课堂内常用操作

### 6.1 翻页与同步

- 底部栏点击“上一页 / 下一页”
- 默认情况下学生端会跟随教师端翻页（可在“课堂设置”中修改）

![图 9：课堂界面底部栏（上一页/下一页/绘制/设置等）（示例）](images/user-guide/teacher/09-toolbar.png)

### 6.2 绘制标注（板书/圈重点）

底部栏点击“绘制”，会出现绘制菜单：

- 工具：钢笔 / 记号笔 / 荧光笔 / 橡皮
- 粗细：滑杆调节
- 颜色：预设色或颜色选择器
- 清空本页：清除当前页的所有标注
- 退出：退出绘制模式

提示：标注会同步到学生端，并在翻页回来后保留。

![图 10：绘制菜单（工具/颜色/粗细/清空）（示例）](images/user-guide/teacher/10-annotate-menu.png)

### 6.3 课堂设置（强烈建议上课前检查）

教师端可打开“课堂设置”调整：

- 强制全屏（学生端上课时自动全屏、限制退出）
- 学生端是否跟随翻页
- 课件缩放（避免内容溢出）
- 学生端卸载管理员密码（用于管理机房环境）

![图 11：课堂设置面板（示例）](images/user-guide/teacher/11-settings.png)

常用建议：

- 机房电脑分辨率/比例不一致时：优先调小“课件内容缩放”
- 如果学生端频繁退出全屏：打开“退出全屏告警”

### 6.4 在线学生与机房视图（可选）

教师端顶部会显示在线学生人数。点击可打开机房视图，用于查看在线状态与学生操作日志。

![图 12：在线学生人数入口与机房视图（示例）](images/user-guide/teacher/12-classroom-view.png)

## 7. 课件导入/导出（不懂编程也能用）

### 7.1 导入课件（.lume）

教师端支持把 `.lume` 课件导入到系统中。导入后刷新课程列表即可看到。

操作步骤：

1. 在教师端“课程选择界面”找到“导入课件”按钮（或菜单入口）
2. 选择一个 `.lume` 文件
3. 等待导入完成提示
4. 点击“刷新课程列表”

![图 13：导入课件按钮位置与导入成功提示（示例）](images/user-guide/teacher/13-import-course.png)

### 7.2 导出课件

可把系统里的课件导出成 `.lume` 文件，用于备份或发给其他老师。

![图 14：导出课件入口（示例）](images/user-guide/teacher/14-export-course.png)

## 8. 常见问题（按这个顺序排查最快）

### 8.1 学生端显示“无法连接到教师端”

按顺序排查：

1. 教师端是否已经打开并在上课界面？
2. 教师机和学生机是否在同一局域网？
3. 学生端配置的“教师机 IP”是否正确？
4. 教师机防火墙是否允许访问端口 `3000`？

如果你需要一个最快的“现场救火”办法：

- 让一台学生机先连上（验证 IP 与网络没问题）
- 再让机房管理员批量同步配置到所有学生机

### 8.2 学生端画面跟不上/卡顿

建议：

- 教师端“课堂设置”中适当调小“课件内容缩放”
- 如果课件包含人脸识别等 AI 页面，机房电脑性能差时可能会变卡（属于正常现象）

# 萤火课堂（LumeSync）学生端使用说明（零基础）

这份说明写给学生、机房管理员、班级信息员：不需要懂编程，照着做即可加入课堂。

---

## 1. 学生端是做什么的？

学生端用于“观看老师演示 + 参与互动”：

- 自动连接教师端课堂
- 课件页面随老师同步（默认）
- 上课期间可被设置为强制全屏，减少分心
- 支持断线后自动重连

![图 1：学生端课堂界面（示例）](images/user-guide/student/01-classroom.png)

---

## 2. 安装前你需要知道的信息

学生端需要知道教师机地址：

- **教师机 IP**：例如 `192.168.1.100`
- **端口**：默认 `3000`（通常不用改）

如果你不知道教师机 IP：

- 问老师/机房管理员
- 或让老师在教师端界面查看“学生端访问地址”

![图 2：老师提供的“学生端访问地址”示例](images/user-guide/student/02-teacher-url.png)

---

## 3. 安装与启动

### 3.1 安装（需要管理员权限）

双击 `LumeSync-Student-Setup-*.exe` 安装包。

注意：

- 安装过程可能会要求管理员权限（这是正常的）
- 安装后学生端会作为 Windows 服务常驻（用于上课管理）

![图 3：学生端安装向导与权限提示（示例）](images/user-guide/student/03-install.png)

### 3.2 启动

安装完成后打开“LumeSync Student / 学生端”。

首次启动如果还未配置教师机 IP，可能会显示“无法连接到教师端”页面。

![图 4：无法连接页面（示例）](images/user-guide/student/04-offline.png)

---

## 4. 连接不上怎么办（最常见）

当你看到“📡 无法连接到教师端”时，按顺序排查：

1. **确认老师端已打开**：老师必须先打开教师端并启动课堂服务
2. **确认同一网络**：学生机与教师机必须在同一局域网（同一个 Wi‑Fi/路由器）
3. **确认教师机 IP 正确**：学校更换教室/更换网段时最容易出错

页面上会显示正在尝试连接的地址，例如：

- `http://192.168.1.100:3000`

如果地址不对，请按下面方式修改。

### 4.1 最快判断：是不是 IP 写错了？

如果你们刚换了教室/刚换了 Wi‑Fi，IP 很可能变了。请直接问老师“教师机 IP 是多少”，以老师当堂显示的为准。

---

## 5. 修改教师机 IP（管理员设置）

在“无法连接”页面下方会提示：

- “如需修改教师机 IP，请右键托盘图标 → 管理员设置”

操作步骤：

1. 在 Windows 右下角托盘找到学生端图标
2. 右键 → 选择“管理员设置”
3. 输入管理员密码（默认密码由老师设置；初始可能是 `admin123`，以老师为准）
4. 填写：
   - 教师机 IP 地址（例如 `192.168.1.100`）
   - 端口号（默认 3000）
5. 点击“保存设置”
6. 返回后等待自动重连或点击“立即重试”

![图 5：托盘图标与右键菜单（示例）](images/user-guide/student/05-tray.png)

![图 6：管理员设置页面（教师机 IP/端口/开机自启）（示例）](images/user-guide/student/06-admin-settings.png)

### 5.1 填写规则（避免常见错误）

- 教师机 IP 只填数字和点，例如：`192.168.1.100`（不要写 `http://`）
- 端口一般保持 `3000`
- 保存后如果仍无法连接，先确认教师端是否开启，再点“立即重试”

## 6. 上课时你会看到什么？

### 6.1 等待页面

当老师还没选课程时，你会看到“等待老师选择课程…”的提示。

![图 7：等待老师选择课程页面（示例）](images/user-guide/student/07-waiting.png)

### 6.2 课堂页面

当老师开始授课后，你会看到课件页面，并跟随老师翻页（默认）。

### 6.3 强制全屏（由老师控制）

如果老师开启了“强制全屏”，上课期间学生端可能会：

- 自动进入全屏
- 退出全屏会被记录/提示

这是为了保证课堂秩序与教学效果。

![图 8：强制全屏提示/课堂顶栏（示例）](images/user-guide/student/08-fullscreen.png)

## 7. 常见问题

### 7.1 我可以关闭学生端吗？

课堂进行中一般不允许关闭；非课堂状态下关闭会最小化到托盘（由学校/老师设置为准）。

### 7.2 我怎么知道自己“连上了”？

一般有三种情况：

- 能看到课堂内容：说明已连上
- 只能看到“等待老师选择课程”：说明已连上，只是老师还没开始
- 一直显示“无法连接”：说明未连上（请回到第 4/5 节排查）

### 7.2 画面卡顿/延迟怎么办？

可能原因：

- 网络信号不稳定（Wi‑Fi 距离太远）
- 电脑性能较弱，遇到人脸识别等 AI 页面会更吃性能

建议：

- 尽量使用有线网络或靠近路由器
- 关闭其它占用性能的软件

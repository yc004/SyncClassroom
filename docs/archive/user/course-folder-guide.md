# 课程文件夹管理功能说明

## 概述

萤火课堂支持将课件按文件夹分类管理，便于教师组织和查找课件。

## 功能特点

1. **创建文件夹** - 自由创建分类文件夹（如：数学、英语、物理等）
2. **删除文件夹** - 删除不需要的文件夹（课件不会被删除，只是移出文件夹）
3. **重命名文件夹** - 随时修改文件夹名称
4. **移动课件** - 将课件在不同文件夹间自由移动
5. **文件夹显示** - 课程选择界面按文件夹分组显示课件

## 使用方法

### 1. 创建文件夹

1. 在课程选择界面，点击右上角「文件夹管理」按钮
2. 点击「新建文件夹」按钮
3. 输入文件夹名称（如：高一数学、英语听力）
4. 点击「创建」或按回车键

### 2. 移动课件到文件夹

有两种方式可以将课件移动到文件夹：

**方式一：通过课件管理器**
1. 点击「课件管理」按钮
2. 使用文件夹下拉框筛选课件
3. 选择课件右侧的「文件夹」下拉框
4. 选择目标文件夹或「无文件夹」移出

**方式二：课程选择界面（自动归类）**
课件管理器中移动课件后，课程选择界面会自动按文件夹显示。

### 3. 重命名文件夹

1. 在「文件夹管理」界面
2. 点击文件夹右侧的「编辑」图标（笔形图标）
3. 修改文件夹名称
4. 点击「保存」或按回车键

### 4. 删除文件夹

1. 在「文件夹管理」界面
2. 点击文件夹右侧的「删除」图标（垃圾桶图标）
3. 确认删除

**注意**：删除文件夹不会删除课件，课件会自动移到「未分类」区域。

## 数据存储

文件夹配置保存在：`public/data/course-folders.json`

文件结构：
```json
{
  "folders": [
    {
      "id": "folder-1234567890",
      "name": "高一数学",
      "icon": "📐",
      "createdAt": "2025-01-15T10:30:00.000Z"
    }
  ],
  "courses": {
    "knn-demo": "folder-1234567890",
    "survey-demo": "folder-1234567891"
  }
}
```

## API 说明

### 创建文件夹

```http
POST /api/course-folders
Content-Type: application/json

{
  "name": "文件夹名称",
  "icon": "📁"  // 可选，默认 "📁"
}
```

### 删除文件夹

```http
DELETE /api/course-folders/:folderId
```

### 重命名文件夹

```http
PUT /api/course-folders/:folderId
Content-Type: application/json

{
  "name": "新文件夹名称",
  "icon": "📐"  // 可选
}
```

### 移动课件到文件夹

```http
PUT /api/course-folders/:folderId/courses/:courseId
```

**参数说明**：
- `folderId`: 目标文件夹 ID，或使用 `null` 移出文件夹
- `courseId`: 课件 ID

## 注意事项

1. **课件不会因删除文件夹而被删除**
   - 删除文件夹只是逻辑删除
   - 课件会自动移到「未分类」区域

2. **文件夹名称建议**
   - 使用有意义的分类名称（学科、年级等）
   - 避免过长名称（建议 10 字符以内）

3. **导入课件后**
   - 新导入的课件默认在「未分类」
   - 需要手动移动到文件夹

4. **刷新课程列表**
   - 移动课件后可能需要刷新才能看到更新
   - 点击「刷新课程列表」按钮
